## Discoveries
- expr-eval library has CVE-2025-12735 (CVSS 9.8 RCE) — replaced with math.js
- Option column stamping approach has a fundamental flaw (tree operations corrupt stamps) — replaced with junction table
- Real-time collaboration via Supabase Realtime is more achievable than expected (~1 extra session, not a massive overhaul) — key insight: the calc engine is already a pure function, so remote edits just become another mutation source
- Item auto-promotion is a genuine differentiator — no existing estimating tool supports this iterative workflow
- Allowance tracking is a missing industry-standard concept — added to item details
- The "single-user" assumption was wrong — Zac has employees and clients who need accounts
